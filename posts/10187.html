go 数组

发表于2021-11-02|go
|阅读量:

Go 语言数组声明有两种方式，1.定长数组，2.不定长度数组
注意：

  1. 如果没有赋值，会有默认值
    • int 数组，默认初始化为 0
    • string 就是空串
  2. 数据类型必须相同，不能混用，int数组不能包含float 等，已验证
  3. boolean 是false
  4. 数组越界是 panic 异常
  5. 数组是值类型，默认情况下是值拷贝，也就是说每次拷贝后是一个独立副本，数组间互不影响。
  6. 可以使用指针来实现引用传递
